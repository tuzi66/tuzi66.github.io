## 第二、如何替换Ubuntu源镜像

这里我们可以通过下面办法替换源镜像。

1、备份已有源

```
    cp /etc/apt/sources.list /etc/apt/sources.list.bak
```

2、编辑源文件

```
 /etc/apt/sources.list
```

然后上上面我们选择的源替换即可。

3、更新系统

```
 apt update -y
```

更新Ubuntu系统，反馈到更新正常没有错误就说明正常替换到国内源镜像，如果有错误需要单独解决。
